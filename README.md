# ai-hands-on

## もぐらたたき!(Whack-a-Mole)

ブラウザで遊べるもぐら叩きゲームです。ビルド不要・外部ライブラリなしの単一HTMLファイルで動作します。

### 遊び方

1. `whack-a-mole.html` をブラウザで開く
2. 「スタート!」ボタンを押す
3. 穴から出てくるもぐらをクリック/タップして叩く(1匹 +10点)
4. 制限時間は30秒。時間が経つほどもぐらの動きが速くなり、終盤は2匹同時に出現します

### 特徴

- **単一HTMLファイル** — HTML/CSS/JavaScript をすべて1ファイルに内包。開くだけで遊べます
- **SVG描画のもぐら** — 表情つきのオリジナルもぐら。叩かれると目が×になります
- **効果音・BGM** — Web Audio API ですべて動的生成(音声ファイル不要)。8bit風ループBGM、ヒット音、カウントダウン音、終了ジングル。右上の 🔊 ボタンでミュート切替
- **ハイスコア保存** — localStorage に記録され、次回プレイ時にも表示。更新時はお祝い演出つき
- **難易度カーブ** — 時間経過で出現間隔・表示時間が自動的に短縮
- **レスポンシブ対応** — PC・スマホどちらでもプレイ可能(`pointerdown` によるタップ遅延なし)

### 動作環境

モダンブラウザ(Chrome / Edge / Firefox / Safari)。サーバー不要でローカルファイルとして開けます。

### 保存されるデータ

| localStorage キー       | 内容             |
| ----------------------- | ---------------- |
| `whackAMole.highScore`  | ハイスコア       |
| `whackAMole.muted`      | ミュート設定     |
